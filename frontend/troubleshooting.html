üí° Common Solutions

üì± Mobile Device Issues

Problem: Files won't transfer between phone and computer

Solution: Mobile networks often block peer-to-peer connections. Our app automatically detects this and uses server relay for reliable transfers.

  • Make sure both devices are connected to internet
  • Wait for "Using server relay" message (this is normal)
  • Large files may take longer on mobile data

üè¢ Corporate/School Network

Problem: Connection failed on work/school WiFi

Solution: Corporate firewalls block WebRTC. We automatically fall back to server relay.

  • This is completely normal on corporate networks
  • Server relay works through standard HTTPS
  • All transfers are still end-to-end encrypted

üè† Different WiFi Networks

Problem: Devices on different WiFi networks can't connect

Solution: Cross-network transfers use server relay automatically.

  • Direct connections only work on same network
  • Server relay handles different networks seamlessly
  • Speed may be slightly slower but very reliable

üîí VPN/Proxy Issues

Problem: Using VPN or corporate proxy

Solution: VPNs change your network setup, but server relay still works.

  • Try disabling VPN temporarily
  • Server relay works through most VPNs
  • Some VPNs block WebRTC entirely

‚ö° Speed Issues

Problem: Transfers are slower than expected

Solutions:

  • Use WiFi instead of mobile data for large files
  • Same network = faster: Both devices on same WiFi
  • Close other apps: Free up bandwidth
  • Server relay: More reliable but may be slower

üÜò Still Having Issues?

If transfers still fail after trying the above solutions:
