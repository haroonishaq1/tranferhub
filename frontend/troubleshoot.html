üîç Common Issues & Solutions

If you see "Connection Failed" error:

  • Different Networks: Make sure both devices are connected to the internet (WiFi/mobile data)
  • Server Relay: The app should automatically use server relay for cross-device transfers
  • Upload Issues: Try refreshing the page and uploading files again
  • Download Issues: Make sure the transfer code is correct and files were uploaded successfully

If upload button doesn't work:

  • Make sure you've selected files first
  • Check if JavaScript is enabled in your browser
  • Try refreshing the page
  • Check browser console for errors (F12 ‚Üí Console)

Cross-device transfer tips:

  • P2P connections often fail between different networks - this is normal
  • The app will automatically fall back to server relay
  • Server relay uploads files to our server temporarily for download
  • Make sure both devices have stable internet connections
